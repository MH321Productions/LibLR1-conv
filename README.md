# LibLR1-conv

This library is a port of [LibLR1](https://github.com/DustStormPettigrew/LibLR1)
(Huge thanks to @DustStormPettigrew and [Will Kirkby](https://bitbucket.org/WillKirkby/liblr1/src/master/))
to C++20, which adds the functionality to save the resources in common-ish formats (like PNG or WAV).
